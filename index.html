Servidor PSCP

Ip: pscp.com.br

Um servidor para Minecraft Bedrock e Java Edition com diversos minigames para competir com os outros jogadores ou com seus amigos, visando sempre a diversão de ambos os lados para que ninguém se sinta mal após uma partida. Aberto a todos os públicos, prometemos entregar uma experiência amigável e desafiadora, desafiando o raciocínio dos jogadores e também seu tempo de reação, porém, tudo de forma que todos possam jogar sem desvantagens.

Como Acessar: O servidor só pode ser acessado em uma versão igual ou superior a 1.12 utilizando o Ip pscp.com.br
